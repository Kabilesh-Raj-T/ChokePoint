# Support Policy

BlastRadius is an open-source project maintained on a best-effort basis.

## Supported Channels

- GitHub Issues for bugs and enhancement discussions.
- GitHub Discussions, if enabled, for usage questions and design discussion.
- Security advisories for private vulnerability reporting.

## What To Include

For bug reports, include:

- BlastRadius version.
- Python version and operating system.
- Minimal input file or topology that reproduces the issue.
- CLI command or API call used.
- Expected and actual behavior.

## Version Support

The maintainers support the latest 1.x release line. Older pre-1.0 versions are
not supported.
